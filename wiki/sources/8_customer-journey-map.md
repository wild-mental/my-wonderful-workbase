# 고객 여정 지도(CJM) 요약

**원본:** `raw/assets/8_customer-journey-map.md`

## 핵심 요약

6종 페르소나별 고객 여정을 5단계(인지→고려→결정→온보딩→충성도)로 상세 매핑. 각 단계의 행동·생각·감정·Pain Point·개선 기회를 체계적으로 정리.

## 페르소나별 여정 핵심 가치

| 페르소나 | 여정 핵심 가치 | 감정 곡선 |
|---|---|---|
| **C1 한정훈** | "40분짜리 스프레드시트 작업을 3초로" | 반신반의 → 비판적 집중 → **즉각 확신** → 루틴 정착 |
| **C2 박소연** | "처음 사는 사람도 30분 안에 확신을 갖고 결정" | 불안·혼란 → 기대 반 불안 반 → **자신감·전도사** |
| **A2 정수빈** | "트렌드 성분이 진짜인지 5초 팩트체크, 결과는 1탭 공유" | FOMO·의심 → 납득 → **만족·공유 충동** |
| **E2 김도현** | "데이터 출처를 2클릭 안에 추적, 오류 48시간 내 수정" | 냉소 → 비판적 검증 → **신뢰 갱신** → 고신뢰 충성 |
| **E1 나경아** | "3초 만에 결론 하나. 자녀와 쉽게 공유" | 의존 → 당혹 or 안도(UX 분기점) → 간접 신뢰 |
| **N1 조미라** | 직접 전환 불가. 자녀 매개 간접 접근만 유효 | 방어·거부 → (대부분 이탈) |

## CJM 단계별 공통 Pain Top 3

| 순위 | Pain Point | 해당 페르소나 | 해결 기능 |
|---|---|---|---|
| **1** | 데이터 신뢰 부재 | C1, E2, A2 | 출처 투명 공개 + 오류 신고 |
| **2** | 성분 해석 불가 | C2, A2, E1, N1 | 성분명 일상어 번역 + 한 줄 해석 |
| **3** | 결정 기준 부재 | C2, A2, N1 | 시장 평균 대비 가격 위치 표시 |

## 플랫폼 성장 플라이휠
A2 바이럴 → C2 유입 → C2 학습(6~12개월) → C1 전환 → C1 CTR → 매출
