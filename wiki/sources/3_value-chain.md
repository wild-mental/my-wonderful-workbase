# 가치사슬(Value Chain) 분석 요약

**원본:** `raw/assets/3_value-chain.md`

## 핵심 요약

건기식 비교 플랫폼의 가치사슬을 설계하고, 각 활동별 경쟁 우위 확보 전략을 정의. 핵심 가치창출 요인으로 **제조원 기반 신뢰 자산**과 **에셋 라이트 수익 구조**를 도출.

## 주요 활동 설계

| 주요 활동 | 설계 방향 | 경쟁 우위 확보 방안 |
|---|---|---|
| **조달 물류** | 식약처 공공 데이터 및 이커머스 가격 API 연동 | **성분-제조원 클러스터링:** '이름만 다른 똑같은 제품' 자동 분류 |
| **운영** | 마케팅 용어 → 표준 성분명 변환 '정제 엔진' | **가성비 스코어링:** 원산지·제조원 숙련도·함량 종합 '리파인 지수' |
| **출하 물류** | 모바일 앱 상시 가격 비교 대시보드 | **최적 장바구니 엔진:** 관세+배송비 포함 '오늘의 실지불가' 자동 산출 |
| **마케팅/영업** | '마케팅 거품 걷어내기' SNS 캠페인 | **투명성 기반 커뮤니티:** 유저 직접 제보+검증 Crowd-sourced 정보 정제 |
| **서비스** | 유료 구독 '성분 상성·부작용 알림' | **전문가 매칭 솔루션:** 정제 데이터 SaaS 제공 |

## 핵심 가치창출 요인 2가지
1. **제조원 기반의 신뢰 자산:** 브랜드 이면의 '진짜 생산자' 정보 공개 → 정보 비대칭성 해소
2. **에셋 라이트 수익 구조:** 직접 재고 보유 없이 정보 중개 + 제휴 수수료 → 물류 리스크 없이 확장
