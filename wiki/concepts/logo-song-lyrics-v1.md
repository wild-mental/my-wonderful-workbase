# Super-Calc 감성 로고송 — 가사 초안

> **장르 감성 노트**  
> 스파스한 신스패드, 조용히 반복되는 비트, 숨 쉬는 공간이 많은 사운드.  
> 보컬은 속삭이듯 시작해서 후렴에서만 조금 열리는 느낌.  
> 가사는 건강기능식품을 직접 언급하지 않는다 — 
> 정보의 소음 속에서 처음으로 조용한 것을 만난 감각을 노래한다.

## 작곡 방향 메모 (AI 작곡 프롬프트용 요약)

| 요소 | 방향 |
|---|---|
| **템포** | 80~90 BPM, 여유로운 4/4 |
| **사운드 레이어** | 신스패드(롱테일) + 킥 없는 리듬 → 후렴에서 서브베이스 등장 |
| **보컬 톤** | 속삭임(verse) → 클린하게 열림(chorus) → 에코(outro) |
| **감정 키워드** | sparse / still / inevitable |
| **피해야 할 것** | 흥분된 업비트, 밝은 팝 코드, 광고 징글 느낌 |
