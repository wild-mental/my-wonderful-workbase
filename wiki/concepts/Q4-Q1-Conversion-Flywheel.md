# Q4→Q1 전환 플라이휠

## 정의

Q4(불안한 탐색자)가 플랫폼을 반복 사용하면서 성분 리터러시가 상승하여 Q1(분석형 탐색자)으로 전환되는 성장 엔진. 이 전환이 사업의 **생사를 가르는** 핵심 메커니즘.

## 전환 경로

```
A2(트렌드 탐색자)
  ↓ SNS 비교 카드 공유
C2 박소연 (Q4-A 건강 계기 진입자)
  ↓ 6~12개월 반복 사용 · 성분 학습
C1 한정훈 (Q1-A 가성비 최적화자)
  ↓ 채널 단가 비교 → 제휴 링크 클릭
💰 수익 발생 (커머스 제휴 수수료)
```

## 전환의 단계 (5단계 설계)

1. **첫 방문:** "비타민D 추천" 검색 → 플랫폼 유입
2. **첫 비교:** 증상 기반 필터로 제품 비교 경험
3. **가격 인식:** "같은 성분인데 가격이 이렇게 다르다니"
4. **학습 축적:** 성분명 일상어 번역을 통해 리터러시 상승
5. **Q1 전환:** "콜레칼시페롤이 뭔지 이제 알겠다" → 직접 비교 시작

## 전환이 발생하면
- 사용자 LTV 상승, 리텐션 개선, 커머스 전환율 상승 → SOM 초과 달성

## 전환이 발생하지 않으면
- Q4 사용자는 1~2회 방문 후 이탈, Q1-A만으로는 규모 제한 → SOM 미달

## PMF 검증 핵심 지표
- MVP 출시 후 **6개월 내 Q4→Q1 전환 사례 5건 이상** 관찰 여부

## 지원 UX 설계
- 이중 레이어 UX: **초보 모드**(핵심 3성분 + 결론) + **전문가 모드**(전체 매트릭스 + 커스텀 정렬)
- 성장한 사용자가 초보 모드만 있으면 이탈 → "이제 전문가처럼 비교해보세요" 트리거
