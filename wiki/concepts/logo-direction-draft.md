# Super-Calc 서비스 메인 로고 방향 정의서

> 지식 베이스(Value Proposition, JTBD, Persona, CJM, Lemon Market 분석) 종합 도출

---

## 🎯 로고가 전달해야 할 핵심 메시지

**서비스의 진짜 차별점은 '기능'이 아니라 '신뢰'다.**

- 건기식 시장은 레몬 마켓 — 소비자는 광고에 지쳐 있고, 믿을 수 있는 정보를 갈구함
- 83%는 가격 차이를 모르고, 47%는 성분 해독 불가 → **혼돈 속에서 명확함을 주는 존재**
- 모든 페르소나 공통 최우려: "이것도 뒷광고 아냐?" → 로고 자체가 **독립성·중립성**을 시각적으로 선언해야 함

---

## 💡 고객 심리 기반 어필 포인트 (JTBD 도출)

| 페르소나 | 고객이 '해고'하는 것 | 로고에서 느껴야 할 것 |
|---|---|---|
| **C1 (가성비 최적화자)** | 엑셀 수작업 + 달러 나눗셈 | ⚡ 속도감, 정밀함, 데이터 엔진의 신뢰성 |
| **C2 (건강 계기 진입자)** | 광고 도배 블로그 + 난해한 성분표 | 🛡️ 안심감, 전문가가 검수한 결론, 따뜻한 확신 |
| **A2 (트렌드 추종자)** | 인플루언서 후회 + 저격 마케팅 의심 | ✅ 팩트 판독기, 공유하고 싶은 브랜드 신뢰 |

> **통합 감정 키워드**: 혼란 → 명확함 / 의심 → 확신 / 피로 → 해방

## 🔤 로고 메시지 방향 (카피 & 슬로건 후보)

### ① 핵심 한 줄 (기존 Value Prop 변형)
> **"광고 없는 팩트, 엑셀 없는 최저가"**
- 장점: 두 가지 핵심 Pain을 한 줄에 담음
- 활용: 로고 하단 서브카피(tagline)로 최적

### ② 속도 + 신뢰 강조형
> **"5초 만에, 광고 없이"**
- 장점: 수치가 즉각적 기대감 형성
- 활용: C1 페르소나를 정조준한 히어로 배너 문구

### ③ 독립성 선언형 (가장 차별화)
> **"협찬 제로. 팩트 100."**
- 장점: 뒷광고 우려를 정면 돌파, E2·C1 신뢰 확보
- 활용: 로고 옆 Badge/인증 스탬프 형태로 병행

### ④ 감성 해방형
> **"복잡한 건강 정보, 여기서 끝내세요"**
- 장점: 탐색 피로(30~60분)에 지친 C2/A2 감성 자극
- 활용: 온보딩 페이지 첫 문장, 부드러운 진입 유도

## 🎨 시각 아이덴티티 방향

### 색상 톤
| 방향 | 의미 | 예시 컬러 |
|---|---|---|
| **신뢰 + 과학** | 식약처·의료 데이터 연상, 냉정한 객관성 | 딥 네이비(#1A2B4C) + 클린 화이트 |
| **투명성 + 중립** | 광고 없음 선언, 유리처럼 투명한 정보 | 아이스 블루(#4A9EBF) + 라이트 그레이 |
| **에너지 + 정밀** | 계산 엔진, 5초 연산, 데이터 파워 | 일렉트릭 그린(#00C49A) 포인트 컬러 |

> ⚠️ **금지**: 따뜻한 오렌지·핑크 계열 → 건기식 광고 브랜드 연상, 중립성 훼손

### 서체 방향
- **숫자·데이터 강조**: Tabular figures 지원 폰트 (예: Inter, DM Mono)
- **한글**: 기하학적, 과도한 장식 없는 고딕 계열 → 정보 명료성
- **혼합**: 영문 로고(Super-Calc) + 한글 슬로건 병기 구조 권장

### 형태 심볼
| 후보 | 의미 |
|---|---|
| 🔲 격자/그리드 | 데이터 정규화, 표준화, 비교의 공정함 |
| ⚖️ 저울/밸런스 | 객관적 비교, 중립성 |
| 🔬 돋보기 + 체크 | 팩트체크, 성분 검증 |
| ✦ 별표/기준점 | "기준이 되는 정보", 북극성 메타포 |

> **추천 조합**: 격자 베이스의 심플한 워드마크(Super-Calc) + 작은 체크마크 배지 심볼

## 📐 로고 시스템 구성 제안

```
[심볼 마크]  [서비스명]
    ✦       Super-Calc
             ——————————————
             광고 없는 팩트, 엑셀 없는 최저가
```

| 버전 | 용도 |
|---|---|
| **풀 버전** (심볼+서비스명+슬로건) | 랜딩 페이지 히어로, 소개 자료 |
| **미니 버전** (심볼+서비스명) | 앱 헤더, 파비콘, SNS 프로필 |
| **배지 버전** ("협찬 제로" 스탬프) | 보고서·팩트체크 결과 옆 신뢰 인증 |

## 🚫 로고 설계 시 절대 금지

지식 베이스 `Lemon-Market-Trust-Design.md`의 **중립성 방어 원칙**을 시각 아이덴티티에도 적용:

- ❌ 제품 사진·캡슐 이미지 → 특정 제조사 연상
- ❌ 의사·가운 이미지 → 의료 행위 오인 소지
- ❌ "최고", "최저가 보장" 등 절대적 수식어 → 객관성 훼손
- ❌ 밝은 광고성 컬러(오렌지, 핑크 그라데이션) → 뒷광고 플랫폼 연상

## ✅ 우선순위 정리

1. **1순위**: `"광고 없는 팩트, 엑셀 없는 최저가"` — 기존 VP를 그대로 tagline으로 활용, A/B 테스트 기준점
2. **2순위**: `"협찬 제로. 팩트 100."` — 차별화 강도가 가장 높음, 뒷광고 시장에서 강렬한 포지셔닝
3. **색상**: 딥 네이비 + 일렉트릭 그린 포인트 조합 우선 탐색
4. **심볼**: 격자 베이스 워드마크 + 체크 배지 병행 시스템
